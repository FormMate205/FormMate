## **1) AR 기반 실시간 환율 변환 서비스**

### 📌 요약

- 해외에서 상품 **가격을 스캔하면 AR로 즉시 변환된 가격**을 보여주고, 환율 및 예산과 관련된 여행에서 유용한 데이터를 표시해주는 서비스

### 📌 주요 기능

**1. AR + OCR 기반 실시간 가격 변환**

- (모바일) 카메라로 상품 가격을 스캔하면 자동으로 가격 인식
- OCR(문자 인식) 활용해 가격 추출 (통화 기호 포함)
- 환율 변환 오픈 API 활용하여 실시간 환율 변환 (ex. 한국수출입은행/하나금융그룹/신한)
- AR 오버레이로 변환된 가격을 상품 위에 실시간 표시

**2. 최적 결제 옵션 추천 + 여행 예산 관련 정보 제공**

**3. 위치 기반 환전소 추천** (부가 기능으로 미룰지 고민 필요)

사용자의 현재 **위치를 기반으로 근처 환전소, ATM** 위치 정보 제공

### 📌추가 고려 사항

- 장바구니 기능:
  - 특정 매장에서 누적하여 금액이 쌓일 경우 고려 필요
- 면세 기능:
  ⇒ 품목 분류까지 들어가야 해서 어렵울 것 같아서 우선 보류

# 2) NFC 기반 스마트 용돈 키링

Finance + (tapping)Ping 금융 + 탭핑

**📌 요약**

**NFC 키링 태깅만으로 아이들이 용돈 관리 & 소비 습관을 관리할 수 있는 스마트 용돈 관리 서비스**

- NFC 태그를 활용해 자연스럽게 금융 습관을 학습
- 앱 설치 없이 웹 기반 동작, 자동 로그인과 보안 기능 제공

### **🔹 핵심 기능**

1️⃣ **NFC 태깅으로 맥락에 맞는 웹 URL 실행** → 자동 로그인(최초 태깅에서만 로그인)

2️⃣ **태깅 맥락에 따라 URL 제공** (아래 설명 참조)

3️⃣ **웹페이지 내부에서 모든 기능 탐색 가능**

4️⃣ **기기 등록 기반 보안 강화** → 다른 기기에서는 자동 로그인 차단

---

- NFC 태그 관련 정보
  ### **구매하여 사용 가능한 NFC 태그 “NTAG216”**
  - **저장 용량: 888 bytes** → URL 외 추가 데이터 저장 가능
  - **UID(고유 ID) 제공** → 태그마다 고유한 UID를 서버에서 활용해 인증 가능
  - **NDEF 표준 지원** → 모든 스마트폰에서 URL 자동 실행
  - **보안 기능** → 암호화된 쓰기 방지 가능
  - **키링 형태의 제품 다수 존재** → 실리콘, 플라스틱 등 다양한 옵션
  - **가격**: 개당 **1,000 ~ 2,000원**

* 위 태그를 기성 키링에 부착하여 사용 가능
