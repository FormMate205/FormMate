# AR 환율 변환 서비스 - 아이디어 구체화

### 1. AR + OCR 기반 실시간 가격 변환

- 사용자가 카메라로 상품 가격을 스캔하면 자동으로 가격 인식
- AR 오버레이로 변환된 가격을 상품 위에 실시간 표시

### 2. 결제 관련 추가 정보 제공

- 카드 결제와 현금 결제 중, 더 유리한 결제 방식을 추천

### 3. 추가 기능 - 현재 회의 중

**AR 기반 장바구니 + 여행 예산과 비교**

- 사용자의 여행 예산을 미리 입력하면 **남은 예산 대비 구매 가능한 상품인지 안내**
- 실시간으로 **소비 내역 추적** 및 장바구니로 관리
- AR 오버레이를 통해 **예산 내에서 적절한 소비인지 안내**
  - 예: "이 물건을 사면 오늘 예산의 80%를 사용하게 됩니다!"

### 4. 추가 고려 사항

- SSAFY에서 제공하는 은행 API를 활용하는 방향으로 디벨롭
- 사용자를 끌어모을 수 있는 기능을 고려해야 함.
- 환전 수수료 등을 고려해야 함.
