# 3월 10일 월요일 회고

### 유저플로우 작성
1. 사용자간의 거래 및 계약서 작성:
    1. 지은이가 동욱이에게 30만원을 빌린 상황 (통화 / 카톡 캡처)
    2. 인공지능이 적절한 계약서 템플릿을 선택하여 초안을 작성해줌
    3. 초안이 완성되면 계약자들에게 초안이 전송되고 카톡방이 생성됨
    4. 사용자가 수정을 원하는 경우 채팅방에서 채팅을 통해 계약서의 수정이 가능하게 됨
    5. 상호 동의가 완료되면 채팅방이 사라지고, 계약이 시작됨

2. 중개 과정

---

### 주요 기능
1. 계약 생성
    1. 음성기반 생성
    2. 챗봇 기반 생성

2. 마이페이지
- 싸피페이 (다른 계좌를 연동하고, 싸피 페이를 관리하는 페이지)
- 입출금 내역 관리
- 신용온도
    - 이전의 거래 내역: 거래한 금액과 기간을 확인할 수 있도록 하면 좋을 것 같음(사기 방지를 위해)
- 계약 일정들을 관리
    - 다른 사람이 갚을 돈에 대한 표시 (데이터를 통해서)
    - 내가 갚아야 하는 돈에 대한 표시
    - 각 계약의 진행도가 어느 정도인지, 납부일이나 납기일이 언제인지 확인하는 가능
    - 각 계약에 대한 송금 / 재촉(?) 버튼 모두 필요
3. 나의 계약 관리
    - 나의 계약서들을 모두 확인 가능한 페이지
        - 계약서를 눌렀을 때 채팅 화면이 나오도록 설정

4. 알림 기능
