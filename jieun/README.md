# AR 기반 실시간 환율 변환 & 최적 결제 추천 서비스

## 📌 요약

- 해외에서 상품 **가격을 스캔하면 AR로 즉시 변환된 가격**을 보여주고, **카드 결제 vs. 현금 환전** 중 가장 유리한 **결제 방법을 추천**하는 서비스
- 환전소 추천, 환율 변동 알림 등 여행자에게 필요한 기능을 함께 제공

---

## 📌 개요

**해외 여행 시 필요한 AR 기반 실시간 가격 변환 앱**

- 스마트폰 카메라로 상품 가격을 찍으면 자동으로 환율 적용
- 실시간으로 해당 가격에 대한 결제 예상 금액 표시
- 다양한 결제 옵션 (카드 결제 vs 현금 환전) 추천

---

## 📌 기대 효과

- 해외에서 복잡한 환율 계산 없이 즉시 가격 확인 가능
- 실시간 최적 결제 수단 추천으로 수수료 절감
