## 기획 구상 & 화면 구상

### 서비스 플로우

1. 통화 음성 녹음
2. 음성 파일 업로드
3. ai 분석 -> 템플릿 선정 -> 자동 계약서 작성(초안 생성)
4. 채팅방 생성
5. 계약서 확인 및 수정
6. 당사자 양방향 서명
7. 계약서 완성
8. 중개 시작 (채팅에서 중간 알림, 송금 내역 등의 알림)

### 화면별 제공 기능

1. 마이페이지

   - 예금/출금 내역
   - 나의 페이 관리
   - 설정 (로그아웃/탈퇴 등)
   - 신용 온도 (금액/기간 등 내역 기반)

2. 계약 생성

   - 음성 기반 생성 (채팅방 생성)
   - 챗봇 기반 생성 (채팅방 생성)

3. 계약 관리

   - 일정 관리 (전체 계약, 클릭하면 각 계약별 상세페이지)
   - 각 일정 세부 관리
     (계약서 확인/수정, 송금 현황, 알림 등...)

4. 채팅
   - 계약별 채팅 내역
