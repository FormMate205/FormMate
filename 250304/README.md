## 기획 구상 & 기능 체크

### AR + OCR 기반 실시간 가격 변환

- (모바일) 카메라로 상품 가격을 스캔하면 자동으로 가격 인식
- OCR(문자 인식) 활용해 가격 추출 (통화 기호 포함)
- 환율 변환 오픈 API 활용하여 실시간 환율 변환 (ex. 한국수출입은행/하나금융그룹/신한)
- AR 오버레이로 변환된 가격을 상품 위에 실시간 표시

##### 1. OCR(Optical Character Recognition) – 상품 가격 인식

- 카메라로 상품의 가격 태그를 스캔하면, OCR을 이용해 텍스트를 추출
- 통화 기호($, €, ¥ 등)까지 분석하여 상품의 원래 가격을 파악
- 추출한 가격 정보를 환율 변환 API와 연동하여 실시간 변환

##### 2. AR(증강 현실) 기반 실시간 가격 변환 표시

- 상품 가격을 카메라로 촬영하면, 변환된 가격을 AR로 상품 위에 오버레이
- **AR 구현 방법**
  1. WebXR API를 사용한 AR 텍스트 오버레이
  2. **React Native Camera + 오버레이** 방식으로 완전한 AR은 아니지만 AR 경험 제공
  3. 핵심 AR 기능만 Swift/Kotlin으로 모듈 개발 -> React Native에서 호출하여 사용
